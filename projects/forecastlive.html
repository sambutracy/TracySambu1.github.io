Detailed Mini Case

ForecastLive — On-Chain F1 Prediction dApp

What it is: A prediction dashboard that stores race predictions on ICP canisters and visualizes scores/leaderboards with a data-driven interface.

Tech stack: Motoko, React, Tailwind CSS, Chart.js, NFID auth, ICP/DFX tooling, Python data service concepts.

My contribution: Built and documented frontend + canister architecture, defined on-chain/off-chain boundaries, and designed scoring visualization experience.

Potential impact: Improves trust and transparency for prediction communities while proving practical Web3 architecture design.
