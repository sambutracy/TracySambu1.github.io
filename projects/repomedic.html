Detailed Mini Case

RepoMedic — AI Self-Healing Repository System

What it is: A flow-first operational system that detects risky code changes, runs validation depth based on risk, and recovers repositories to healthy checkpoints.

Tech stack: TypeScript, Express, React, Vite, Tailwind CSS, GitLab API, GitHub API, Google GenAI.

My contribution: Designed the orchestration model (detect → assess → validate → recover → document), integrated Git platform hooks, and authored maintainability documentation.

Potential impact: Reduces manual firefighting, improves reliability of AI-assisted code workflows, and preserves auditable incident memory in-repo.
